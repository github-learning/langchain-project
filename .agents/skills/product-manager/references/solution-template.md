# 方案设计模板

## 方案选择框架

| 方案类型 | 适用场景 | 决策权重 |
|---------|---------|---------|
| **自研** | 核心业务、高定制、长期维护 | 质量优先 |
| **引入开源** | 通用功能、成熟方案 | 成本优先 |
| **引入商业** | 高可靠要求、低维护能力 | 省心优先 |

## 方案评审检查清单

**技术可行性**
- [ ] 团队有相关技术能力吗？
- [ ] 技术方案经过验证了吗？
- [ ] 有什么技术风险？

**业务匹配度**
- [ ] 方案能完全满足需求吗？
- [ ] 有功能遗漏吗？
- [ ] 有过度设计吗？

**成本效益**
- [ ] 开发成本多少？
- [ ] 维护成本多少？
- [ ] 收益大于成本吗？
